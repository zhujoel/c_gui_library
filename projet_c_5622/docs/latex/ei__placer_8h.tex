Configures the geometry of a widget using the \char`\"{}placer\char`\"{} geometry manager. If the widget was already managed by another geometry manager, then it is first removed from the previous geometry manager. If the widget was already managed by the \char`\"{}placer\char`\"{}, then this calls simply updates the placer parameters\-: arguments that are not N\-U\-L\-L replace previous values. When the arguments are passed as N\-U\-L\-L, the placer uses default values (detailed in the argument descriptions below). If no size is provided (either absolute or relative), then the requested size of the widget is used, i.\-e. the minimal size required to display its content.
